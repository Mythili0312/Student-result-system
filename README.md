# Student Result System

A simple full-stack web application to calculate student results.

## 🚀 Features
- Enter student name and marks
- Calculates total, average, and grade
- Displays result instantly

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- API Communication: Fetch API

## 📌 How it works
1. User enters marks in the frontend
2. Data is sent to Flask backend using POST request
3. Backend processes the data
4. Result is returned as JSON and displayed

## ▶️ Run Locally
1. Install dependencies:
   pip install flask flask-cors

2. Run the server:
   python app.py

3. Open index.html in browser
